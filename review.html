문제 및 해결

1. translate과 rotation 동시 사용
div에 rotation을 넣고 translate을 넣으니 rotation이 적용 안되는 문제가 있었다.
해결법은 translate 옆에 rotation을 넣어서 해결

2. 비디오 배경 문제
백그라운드를 비디오 태그로 대체 했더니 다음과 같은 문제가 발생

  1. fixed로 고정시킨 헤더가 비디오 밑에 파묻히는 현상
  2. 클릭시 움짤이 정지됨
  3. 비디오 위의 글자가 반투명하게 나옴

해결 방안

  1. 헤더에 z-index 부여
  2. video 속성 중 controls 제거

3. 그래프 바 구현
2개의 div를 만들고 하나는 width 100%
다른 하나는 수치%만큼의 width을 주고
겹치는 것으로 구현하러 했으나 겹쳐지지 않는 문제

해결 방안

barhead에 position:relative 를 주는게 아닌
우측처럼 부모 자식으로 묶고 width 100%를 줄 div에 주는 것으로 해결

4. list style : none 상태에서 list style 추가시 추가 안됨
display: list-item;
list-style-type:을 넣어서 해결
